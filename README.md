Exemples de données : 

[Tailles]
LibelleCourt=11
Prix=13
Provenance=8
Poids=12
Colonnes=2
Lignes=9
MaxCharsLibelle=30

Pour charger un fichier CSV il suffit de cliquer sur CSV
Il doit impérativement avoir les colonnes suivantes :

Libellé court;Référence externe;Prix de vente;Prix soldé;Code-barres;Poids
